Everything starts from the conjugation map 
$$(g,h) \to ghg^{-1}$$
Thought of as a map $Ad(g):G\to G$.

For any particular $g$, this defines an diffeomorphism on $G$ which has [[Differential|differential]] 
$Ad_G \in \GL(\mathfrak{g})$. We can thus imagine that $Ad_G$ is a function $G\to \GL(\mathfrak{g})$.

This, too, has a differential, which is thus called $d(Ad_G(g)) = Ad_{\mathfrak{g}}: \mathfrak{g}\to \GL(\mathfrak{g})$.
